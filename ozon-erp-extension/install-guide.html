Ozon ERP Chrome 插件安装指南

在 Ozon 商品页面一键计算利润、快速上架

1下载插件安装包

点击下方按钮下载插件 zip 压缩包：

下载 Ozon ERP 插件 (ozon-erp-extension.zip)

下载完成后，将 zip 文件解压到电脑上任意位置（如桌面），得到一个文件夹。

2打开 Chrome 扩展管理页

在 Chrome 浏览器地址栏输入以下地址并回车：

chrome://extensions

打开页面右上角的「开发者模式」开关。

3加载插件

点击页面左上角「加载已解压的扩展程序」按钮。

在弹出的文件夹选择窗口中，选择第 1 步解压出来的文件夹（里面有 manifest.json 文件的那个）。

提示：看到扩展列表中出现「Ozon ERP」卡片就说明安装成功了。

4固定到工具栏

点击 Chrome 右上角的拼图图标，找到「Ozon ERP」，点击右侧的固定按钮。

固定后插件图标会常驻在地址栏右侧，方便随时使用。

5登录并使用

点击工具栏的 Ozon ERP 图标，输入你的 ERP 账号密码登录。

还没有账号？先到 erp.aiseo114.com 注册，再回来登录。

登录后访问 Ozon 商品页面（seller.ozon.ru），插件会自动在页面右侧显示利润计算面板，支持：

  • 实时利润率计算（采购成本、运费、佣金）
  • 体积重计算与计抛提醒
  • 一键上架到 ERP 系统
注意：开发者模式加载的插件，Chrome 每次启动可能弹窗提示「禁用开发者模式扩展程序」，点击取消即可继续使用。
