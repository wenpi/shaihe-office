Ozon ERP Chrome 插件安装指南

在 Ozon 商品页面一键计算利润、快速上架

1下载插件文件

点击下方链接下载插件压缩包，解压到本地任意目录：

https://github.com/wenpi/shaihe-office/tree/main/ozon-erp-extension

或者直接克隆仓库：

git clone https://github.com/wenpi/shaihe-office.git

插件文件位于 ozon-erp-extension/ 目录下。

2打开 Chrome 扩展管理页

在 Chrome 浏览器地址栏输入：

chrome://extensions

打开右上角的「开发者模式」开关。

3加载已解压的扩展程序

点击左上角「加载已解压的扩展程序」按钮。

选择刚才下载的 ozon-erp-extension 文件夹（包含 manifest.json 的那个目录）。

提示：如果是从 GitHub 克隆的仓库，选择 shaihe-office/ozon-erp-extension 目录。

4固定插件到工具栏

安装成功后，点击 Chrome 右上角的拼图图标（扩展程序），找到「Ozon ERP」，点击📌固定。

这样插件图标会常驻在工具栏，方便随时使用。

5登录 ERP 账号

点击工具栏的 Ozon ERP 图标，在弹出窗口中输入你的 ERP 账号密码登录。

ERP 后台地址：

https://erp.aiseo114.com
提示：如果还没有账号，先在 ERP 后台注册，然后在插件中登录。

6开始使用

登录后，访问任意 Ozon 商品页面（seller.ozon.ru），插件会自动在页面右侧注入利润计算面板：

  • 输入采购成本、运费等，实时计算利润率
  • 支持体积重计算和计抛提醒
  • 一键上架到 ERP 系统
注意：此插件为开发者模式加载，Chrome 每次启动可能提示「禁用开发者模式扩展程序」， 点击取消即可继续使用。
