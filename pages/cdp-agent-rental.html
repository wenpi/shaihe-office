核心原理

CDP Agent 是一个轻量级 Node.js 客户端（< 10MB），通过 Chrome DevTools Protocol 连接本地已打开的 Chrome 浏览器。它不是模拟器，而是直接操控真实浏览器——使用已登录的账号，像真人一样点击、输入、滚动、截图。

安全约束 — 号主权益保护
多层校验确保账号安全

无需密码

号主只需挂机运行 CDP Agent，不需要给出任何账号密码

操作白名单

号主自定义 allowed_tasks，只允许指定类型的操作

日限额控制

max_daily_tasks 限制每日最大任务数，防止过度使用

操作截图

每个任务执行后自动截图存档，号主可随时查看操作记录

禁止敏感操作

改密码、绑定手机、注销等敏感操作不在任务类型中

紧急下架

号主可随时暂停出租，CDP Agent 离线自动下架
